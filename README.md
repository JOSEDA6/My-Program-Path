# 📝 工单汇总工具（Order Merge Tool）

这是一款基于 **Python** 和 **PySide6** 开发的桌面端工具，专门用于将多个格式杂乱的 Excel 或 CSV 询价单、工单汇总成一个标准化的 Excel 表格。

✨ **告别手动复制粘贴，一键完成数据清洗与对齐！**

------

### 🌟 核心功能

- **🔍 智能识别**：自动探测表头（品名、规格、数量、单价等），兼容中英文多种关键字映射。
- **🧹 数据清洗**：自动统一单位（如：个/pcs -> 个），规范金额与数量的数字格式。
- **🚫 自动去重**：基于单号、品名和规格自动过滤重复行，确保汇总数据唯一性。
- **🎨 专业输出**：生成的 Excel 拥有精美的边框、居中对齐、加粗表头以及自动货币格式。
- **🚀 高效并行**：采用多线程处理技术，处理大量文件时界面依然流畅不卡顿。

   巴什


   巴什

   ```
   pip install PySide6 pandas openpyxl
   ```

------

### 📖 使用方法

1. **启动程序**： 直接运行 。`python your_script_name.py`

2. **添加文件**：

   点击 **[添加 Excel/CSV]** 按钮，支持多选不同路径下的文件。

3. **设置路径**：

   点击 **[修改保存位置]** 选择汇总后的文件存放目录（默认保存至桌面）。

4. **开始任务**：

   点击 **[开始合并任务]**，进度条将实时显示处理进度。

5. **查看结果**：

   完成后程序会自动弹出成功提示，并**自动打开**结果所在的文件夹。📂
